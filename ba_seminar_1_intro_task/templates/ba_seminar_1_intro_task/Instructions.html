Im folgenden sehen Sie Bilder von zwei kreative Ideen. Diese wurden von Innovatoren (TeilnehmerInnen in einem vorhergehenden Experiment) angefertigt. Die Innovatoren hatten die Aufgabe sich einen Begriff auszudenken und diesen mit vorgegebenen Materialien darzustellen.

This task consists of 18 rounds. In each round, you will have to choose between two illustrations provided by two different creators. You choose an illustration by clicking on the button, indicating the respective illustration. {% if player.treat == 'no_info' or player.treat == 'idea_only' or player.treat == 'verbal_only' or player.treat == 'verbal_idea'%} If the creator of the illustration is female this button is purple. If the creator of the illustration is male this button is blue. {% endif %} The creators are workers from another task who were asked to illustrate words (e.g., objects, items or actions) using a fixed set of materials.

It is your goal to choose the illustrations with the highest value. The value of an illustration is determined as follows:

  • Value of illustration = 0 if it is not original
    • An illustration is original if no other creator among 50 randomly drawn creators in this task illustrated the same word.
  • Value of illustration = recognition rate (from 0 to 100) if it is original
    • The recognition rate is the fraction of people (from 0-100%) that recognized the word when we asked them to guess the illustrated word based on the illustration only.

Your bonus

To determine whether or not you receive the bonus, we will randomly choose one out of the 18 rounds after you completed the task.

  • If you chose the illustration with the higher value in the selected round, your bonus is $1.50.
  • If you chose the illustration with the lower value in the selected round, your bonus is $0.
  • If both illustrations have the same value in the selected round, your bonus is $0.75.
